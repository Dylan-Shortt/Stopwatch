# Stopwatch

A simple stopwatch app for Android built with Kotlin.

## What it does

Just a straightforward stopwatch that does what you'd expect. Start it, stop it, reset it. No fancy features, just clean time tracking.

## Features

- Start/Stop button to control the timer
- Reset button to go back to zero
- Clean, simple interface
- Built with Kotlin

## Getting Started

Clone the repo and open it in Android Studio:

```bash
git clone https://github.com/Dylan-Shortt/Stopwatch.git
```

Then just build and run it on your device or emulator.

## How to use

Pretty straightforward:
1. Hit **Start** to begin timing
2. Hit **Stop** to pause
3. Hit **Reset** to go back to 00:00:00
